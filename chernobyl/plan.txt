ЗАДАЧА 1 — ЛОГИ В ТЕРМИНАЛЕ (применяю)
──────────────────────────────────────────────────────────
  Что делаю:
    1. По умолчанию рамка баннера рисуется ASCII-символами
       ( + - | ), а не Unicode box-drawing ( ╔ ═ ║ ).
       Unicode возвращается через FANCY=1 или FORCE_COLOR=1.
    2. Цвета по умолчанию ВЫКЛЮЧЕНЫ.
       Включаются через FORCE_COLOR=1 (или FANCY=1).
       NO_COLOR=1 выключает в любом случае.
    3. Поддержка NO_BOX=1 — простые строчки без рамки.
    4. Декоративные глифы (✓ ⚠ ⏳ ⓘ ↳) при отсутствии FANCY
       заменяются на ASCII ([OK] [!] [...] [i] -> ).
    5. Дублирующий ранний баннер «… презентация запущена»
       подавляется при работе через туннель.
       Остаётся ОДИН большой публичный баннер.
       При NO_TUNNEL=1 печатается локальный баннер
       (как раньше, единожды).

  Файл:
    chernobyl.py
      • _USE_COLOR / _FANCY / _USE_BOX / _USE_BOX_UNICODE
      • _BOX (словарь символов)
      • _GLYPH (словарь глифов)
      • _print_banner       — рамка через _BOX
      • _print_public_banner — рамка через _BOX, глифы через _GLYPH
      • main()              — _print_banner только при NO_TUNNEL

       audio.pause() сразу после прайминга на слайде 0

    2) Cloudflare-ссылку (и резервную lhr.life, если есть)

  Что удалено:
    • строка `[boot] N assets in memory, port=...`
    • строка `[tunnel] downloading cloudflared from <URL>`
        → заменена на короткую «Скачиваю cloudflared (один раз)...»
    • строки `[tunnel] launching: <длинная команда>` (cf и lhr)
    • весь поток `[cloudflared] ...` — десятки строк диагностики
        INF/WRN — теперь читается тихо, парсится URL и всё
    • весь поток `[lhr] ...` — то же самое для localhost.run
    • строка `[tunnel] skipped (NO_TUNNEL=1)` (просто бесполезная)
    • строка `[tunnel] ssh не найден...` — резерв тихо не запускается
    • строки `[tunnel] continuing without a public URL`
    • большая «громкая» рамка `_print_public_banner`
        (статус ССЫЛКА ГОТОВА, big_url на жёлтом фоне,
         clipboard-блок, troubleshooting-блок 1/2/3/4)
    • второй пост-фактум блок «✓ ССЫЛКА ОТВЕЧАЕТ из интернета»
        / «⚠ ССЫЛКА за 2 мин не ответила»
        — probe больше не отчитывается в stdout
    • префикс `[http] ` у HTTP-логов (формат всё равно очевиден)
    • строка `[shutdown] bye` — оставлено просто `bye`
