# chernobyl-prez

Монолитная Python-презентация **«Чернобыльский след в истории моей семьи»**
для запуска в Termux на телефоне.

Один файл `chernobyl.py` содержит:

- Встроенные слайды (base64-JPG, 14 шт.)
- Встроенный аудио-трек озвучки (mp3)
- HTTP-сервер на чистом stdlib
- Автозапуск Cloudflare-туннеля (квик-туннель), при отсутствии — резерв
  через `*.lhr.life` (нужен `openssh`)

## Запуск в Termux

```bash
# 1. Зависимости (один раз)
pkg update -y
pkg install python git openssh -y

# 2. Клонировать
git clone https://github.com/animebyst07-stack/chernobyl-prez.git
cd chernobyl-prez

# 3. Запустить
python chernobyl.py
```

После старта в терминале появится баннер с публичной ссылкой —
открой её в браузере телефона, нажми **«Запустить презентацию»**.

## Переменные окружения

| Переменная       | По умолчанию | Что делает                                              |
|------------------|--------------|---------------------------------------------------------|
| `PORT`           | `5080`       | Порт локального сервера                                 |
| `HOST`           | `0.0.0.0`    | Адрес локального сервера                                |
| `NO_TUNNEL=1`    | —            | Не запускать публичный туннель                          |
| `FANCY=1`        | —            | Включить Unicode-рамку и цвета в логах (как раньше)     |
| `FORCE_COLOR=1`  | —            | То же, что `FANCY=1` (стандарт de-facto)                |
| `NO_COLOR=1`     | —            | Принудительно отключить цвета                           |
| `NO_BOX=1`       | —            | Печатать баннер без рамки, простыми строчками           |

Пример без туннеля и в простом ASCII-режиме:

```bash
NO_TUNNEL=1 python chernobyl.py
# открой http://127.0.0.1:5080/ на этом же телефоне
```

## Управление в браузере

- Слайд 1 — статичная обложка, не уезжает сама.
- Свайп вправо или стрелка `→` — переход к слайду 2 (стартует озвучка).
- Слайды 2–4 переключаются автоматически под аудио.
- С 5-го слайда — ручное управление: `← →`, пробел, свайп, кнопки внизу.
- `F` — полноэкран, `M` — звук вкл/выкл.

## Файлы
